Image Name
The name of the image.
Image Description
A description of the image.
Source Type
Uploading the image from a local file or from a specified URL.
Location
If uploading the image via a HTTP URL the image location must be accessible to the Image Service and be a URL direct to the image binary.
Format
Select the format of the disk from the drop down menu.
Kernel
The name of image stored in the Image service that should be used as the kernel when booting an AMI-style image.
Ramdisk
The name of image stored in the Image service that should be used as the ramdisk when booting an AMI-style image.
Architecture
The CPU architecture that must be supported by the hypervisor. For example, x86_64, arm, or ppc64.
Minimum Disk
Amount of disk space in GB that is required to boot the image.
Minimum RAM
Amount of RAM in MB that is required to boot the image.
Image Visibility
The access permission for the image.
Protected Image
If set to 'Yes' then the image cannot be deleted.
