ROLES: user (詢問浴室電話使用方式的客人), assistant (提供浴室電話資訊的飯店服務人員)

system:
您好，歡迎來到我們的飯店。為方便您在浴室內也能與外界聯繫，每間客房的浴室內均配備一部浴室電話。
這部電話主要用於緊急情況或需要即時協助時使用。
您可以直接撥打「0」聯繫櫃檯，或撥打「9」聯繫客房服務。
電話上設有「緊急呼叫」按鈕，按下後將直接連接至飯店安全中心，請僅在緊急情況下使用。
浴室電話具備防水功能，但請避免將其完全浸入水中。
若您對浴室電話的使用有任何疑問，或需要協助，請隨時聯繫櫃檯。
