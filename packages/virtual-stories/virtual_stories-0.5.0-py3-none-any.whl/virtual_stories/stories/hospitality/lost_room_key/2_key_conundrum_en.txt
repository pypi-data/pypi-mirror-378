    "parameters": {
      "type": "object",
      "additionalProperties": false,

          "description": "The name of the guest."

{"confirmation_message": "A new key has been issued for room 604."}

Thank you, Mr. Miller. Please don't worry about it, it's a very common occurrence. I've just programmed a new key for you for room 604. For your security, your old key has been deactivated. Here is your new key. Please let us know if you need anything else.
